
# Autenticação e Autorização da Equipe Multidisciplinar

**COMO** servidora/membro da equipe multidisciplinar do NAPNE,  
**QUERO** acessar o Acolhe+ com minhas credenciais do SUAP,  
**PARA** gerenciar as informações dos alunos que precisam ser assistidos.

## Critérios de Aceitação

1. O login deve ser realizado via integração SUAP, identificando o papel (vínculo).    
1.1 Se for o primeiro acesso, o sistema deve criar o perfil no banco de dados do Acolhe+, importando Nome Completo, Matrícula, E-mail Institucional e Campus.  
1.2 Caso não esteja associado ao NAPNE e faça parte da equipe, o usuário deverá solicitar ao Administrador do Sistema o acesso ao NAPNE.  
2. Diferente do professor (que vê diários), o servidor que faça parte da equipe deve ser direcionado para um painel com:  
2.1 Barra de busca de alunos.  
2.2 Lista de "Pendências de Validação" (alunos indicados por colegas que aguardam seu "check").  
2.3 Resumo de alunos já ativos sob sua responsabilidade.
